FAQ

Good to know

Your number is sent to the bot backend only to request a pairing code from WhatsApp. The website does not log or persist it.

WhatsApp pairing codes are short-lived. If yours expires before you finish linking, just tap "Request a new code" for a fresh one.

Double-check the number format (it must include your country code), confirm the bot server is online, and try again. If it persists, reach out on Telegram.

Each deployed bot instance links to a single WhatsApp account. To link a second number, deploy a separate instance of the bot.

The console flow still exists and can be re-enabled with an environment variable. The website simply gives you a friendlier default.
